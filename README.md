# Blog CMS

ブログ管理システム（フロントエンド + バックエンド）

## 概要

記事の作成・編集・削除ができる管理画面付きブログシステムです。

## 技術スタック

### フロントエンド
- React
- TypeScript
- Vite (またはCreate React App)

### バックエンド
- Spring Boot(Java)
- MySQL / PostgreSQL

## プロジェクト構成
```
blog-cms/
├── frontend/     # フロントエンド（React + TypeScript）
├── backend/      # バックエンド（Spring Boot）
└── README.md
```
